# Stealth Calculator Web v2 (GitHub Pages)

A modern, rainforest-themed stealth calculator web app designed for private self-control.

## Deploy (phone friendly)
1. Upload the contents of this zip to your GitHub repo root
2. Repo Settings → Pages → Deploy from branch: `main` / folder: `/(root)`
3. Open the Pages link in Safari → Share → Add to Home Screen

## Secret unlock
- Long-press `0` ~1 second → PIN → Notes

## Notes about YouTube rain audio
This version uses in-browser WebAudio for rain. Directly using a YouTube link as an audio source is unreliable (and usually blocked).
If you have a direct `.mp3` or `.wav` URL you control, you can swap it in later.
