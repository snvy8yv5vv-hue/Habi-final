# Stealth Calculator (Web / GitHub Pages)

This is a **single-file** web app that looks like a calculator but hides a private reset space.

## How to deploy (phone-friendly)
1. Create a GitHub repo
2. Upload the contents of this zip (keep the folder structure)
3. Repo Settings → Pages → Deploy from branch: `main` / folder: `/(root)`
4. Open the Pages link in Safari → Share → Add to Home Screen

## Secret unlock
- Long-press `0` for ~1s **OR**
- Tap the display 7 times quickly

## Notes
- PIN + logs are stored in `localStorage` (local to your device).
- Rain audio is generated with WebAudio (no audio files needed).
